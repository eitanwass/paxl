Paxl XML Parser Comparison

Paste (or upload) XML and compare parsing speed between Paxl and TXML, live in your browser.

Each parser is warmed up and run repeatedly so timings reflect steady-state performance, not one-off JIT noise. RapidX2J isn't included here - it's a native Node.js addon with no browser build, so it can only be benchmarked in Node. See benchmarks/benchmarks.js for the full 3-way comparison.
